# Usage
Clone and install dependencies.
```
git clone https://github.com/devinleejones/devinleejones.github.io.git
cd devinleejones.github.io/
npm install
```
Create an '.env' file in the project root.
```
PORT=3000
```
Run the application
```
npm run watch
```
